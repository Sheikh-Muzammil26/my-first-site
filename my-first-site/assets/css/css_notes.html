My first CSS website

CSS এর জন্য কম্পক্ষে ৪ টি জিনিস লাগবে। প্রপার্টি তথা বৈশিষ্ঠ ঃ ভেলু তথা মান property : velue; এখানে প্রপার্টির পরে একটি কোলন দিতে হবে। এরপরে সেমিকোলন। সেমিকোলন দেয়ার মানেই হচ্ছে, এরপরে আরো কিছু চাইলে আমরা দিতে পারব। যেমনঃ color: red; normally color বলতে বুঝায় টেক্সট কালার। আর ব্যাকগ্রাউন্ড-কালার হচ্ছে পিছনের কালার। inline মানে লাইনের মধ্যে। enternal or embeded

px : যতটুকু উচ্চতা বা প্রস্থ তথা সাইজ দেয়া হবে ততটুকুই নিবে। এবার স্ক্রিন সাইজ কমুক বা বাড়ুক। % দিয়ে সাইজ দিলে ফুল স্ক্রিন অনুযায়ি ছোট বড় হবে। যেমন, আমি যদি একটি বক্সের দৈর্ঘ্য দেই ৫০% , তাহলে স্ক্রিন যত ছোটই হোক এর পঞ্চাশ থাকবে বক্সটি।

CSS এ এক্সটার্নাল এবং ইনাটার্নাল স্টাইল করার জন্য style ট্যাগের ভিতরে সিলেক্ট করে করে সাজানো যায়। সিলেক্ট করার নিয়ম হলো- আইডি সিলেক্ট করতে হ্যাশ ইউজ করা হয়।(id হবে একটি ইউনিক নাম।) যেমনঃ #ID_name class সিলেক্ট করতে ডট দিতে হয়। একই ক্লাস একাধিক জায়গায় ইউজ করা যায় কিন্তু একই আইডি একাধিক জায়গায় ইউজ করা যায় না । যেমনঃ .class_name আইডির সাথে সাথে ক্লাসও ইউজ করা যায় একই এলিমেন্টের ভিতরে । যেমনঃ

element সিলেক্ট করার জন্য শুধু ট্যাগ নেইম লিখলেই হয়। যেমনঃ h1{}

tag : opening and clossing মিলিয়ে যা হয় তাকে ট্যাগ বলে। element : একটি ট্যাগ, তার ভিতরের এট্রিবিউট এবং কন্টেন্ট সবকিছুকে একসাথে বলা হয় এলিমেন্ট।

universal: সবকিছুকে একসাথে সিলেক্ট করতে চাইলে শুরুতে * চিহ্ন দিতে হয়। যেমনঃ *body {} আইডি ও ক্লাসের মধ্যে পার্থক্য কী - ইন্টারভিউ তে জিজ্ঞেস করে। group selector: একসাথে দুটি বিষয় (id,tag,class,element) সিলেক্ট করতে চাইলে মাঝে কমা চিহ্ন দিয়ে দিলেই চলবে যেমনঃ h1,h2 {}
